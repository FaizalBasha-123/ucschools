# Student API Routes

Base: `/api/v1/student`

## Profile
- `GET /student/profile` - Student profile
- `GET /student/timetable` - Class timetable

## Attendance
- `GET /student/attendance` - Attendance records

## Academic
- `GET /student/homework` - Homework list
- `GET /student/quizzes` - Available quizzes
- `GET /student/performance` - Academic performance
- `GET /student/reports` - Report cards

## Fees
- `GET /student/fees` - Fee details & payment history

## Communication
- `GET /student/messages` - Class messages (read-only)
- `GET /student/events` - Event calendar
- `GET /student/feedback` - Teacher feedback

## Leaderboard
- `GET /student/leaderboard` - Class rankings

## Transport
- `GET /student/bus-route` - Assigned bus route & stops
