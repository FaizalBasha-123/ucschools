- **Language**: Go 1.25.0
- **Framework**: Gin 1.11.0
- **DB Driver**: pgx 5.6.0 (PostgreSQL)
- **Auth**: golang-jwt 5.2.1

- **MongoDB**: mongo-driver 1.15.0
- **Messaging**: NATS 1.49.0

- **React**: 19.2.3
- **TypeScript**: 5.x

- **Forms**: React Hook Form 7.70.0
- **Validation**: Zod 4.3.5
- **Charts**: Recharts 3.6.0
