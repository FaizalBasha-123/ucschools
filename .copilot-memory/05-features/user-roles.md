# User Roles & Permissions

## Roles
1. **super_admin** - Platform administrator
2. **admin** - School administrator
3. **teacher** - Faculty member
4. **student** - Student
5. **staff** - Non-teaching staff
6. **parent** - Parent/guardian

## Role Capabilities

### Super Admin
- Manage all schools
- User suspend/delete (any role)
- Blog management
- Global catalog (classes, subjects)
- Question/material uploads (shared across schools)
- Demo requests & help center
- System settings
- Trash/recovery

### Admin (School)
- School dashboard & analytics
- User management (within school)
- Student/teacher CRUD
- Class management
- Fee structures
- Attendance oversight
- Bus route management
- Admission approval
- Inventory tracking
- Compliance/consent management

### Teacher
- Class dashboard
- Timetable view
- Homework assignment (with attachments)
- Quiz creation & scheduling
- Attendance marking (bulk CSV upload)
- Question paper generation
- Study materials upload
- Class messaging (WebSocket)
- Salary & fee view
- Student performance reports
- Live whiteboard/virtual classes

### Student
- Personal dashboard
- Timetable view
- Attendance tracking
- Homework submission
- Quiz attempts
- Performance reports & grades
- Fee tracking & payment
- Study materials access
- Event calendar
- Teacher feedback
- Leaderboard view
- Bus route tracking
- Class messages (read-only)

### Staff
- Limited access to specific modules
- Inventory management
- Event management

### Parent
- Child's academic progress
- Fee payment
- Consent management
- Communication with teachers
