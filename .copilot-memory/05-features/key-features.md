# Key Features

## Academic Management
- **Timetables**: Class schedules, multi-teacher support
- **Homework**: Assignment with file attachments, due dates
- **Quizzes**: Timed/anytime, multi-class, auto-grading
- **Assessments**: Term exams, unit tests, breakdowns
- **Question Papers**: AI-generated, difficulty levels
- **Study Materials**: Teacher/super-admin uploads
- **Grades & Reports**: Performance tracking, trends

## Attendance
- Daily marking (per student)
- Bulk CSV upload (teachers)
- Monthly/yearly reports
- Absence notifications
- Trend analysis

## Fee Management
- Fee structures (per class/grade)
- Payment tracking (Razorpay integration)
- Waiver management
- Overdue alerts
- Collection reports

## Transport
- GPS tracking (real-time via WebSocket)
- Route management with stops
- Driver tracking (background geolocation)
- Bus location history
- Stop assignment to students

## Communication
- **Teacher Messaging**: Class groups (WebSocket)
- **Support Tickets**: Public + authenticated
- **AI Chatbot**: Student query resolution
- **Push Notifications**: FCM (mobile + web)

## Admissions
- Public forms (embeddable iframes)
- Auto-approval option
- Application tracking
- Email notifications

## Compliance
- Parental consents (photo/video/data-processing)
- GDPR data subject requests
- Consent audit logs
- Withdrawal tracking

## Interoperability
- DIKSHA integration
- DigiLocker verification
- ABC (Academic Bank of Credits)
- Student transfer requests
- Learner registry sync

## Reports & Analytics
- Attendance trends
- Fee collection stats
- Performance leaderboards
- Revenue charts
- Event calendars

## Mobile Support
- Capacitor-based Android app
- Background geolocation (drivers)
- Offline-first UI
- Push notifications
- Camera/gallery access
